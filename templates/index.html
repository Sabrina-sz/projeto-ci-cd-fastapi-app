Bem-Vindo(a) ao FastApi

Vamos entender um pouco como foi feito para rodar está aplicação

Esta aplicação foi projetada para ser executada automaticamente em um cluster Kubernetes, junto com algumas tecnologias importantes necessárias para o desenvolvimento, deploy e monitoramento:

Tecnologias ultilizadas na projeção da aplicação

  • Ubuntu: Distribuição Ubuntu(linux) no windows para executar os comandos

  • GitHub Actions: Automatização com o build da imagem Docker, faz push para o Docker Hub e atualiza o repositório de manifests Kubernetes.

  • Rancher Desktop: Plataforma para gerenciar o cluster Kubernetes.

  • Repositório de manifests: Contém os arquivos YAML para deploy no Kubernetes (deployment e service).

  • ArgoCD: Monitora o repositório de manifests e sincroniza o estado desejado para o cluster Kubernetes.

  • FastAPI: Framework web usado para criar a aplicação que serve páginas HTML dinâmicas.

  • DockerHub: Armazena e distribui as imagens Docker geradas.
